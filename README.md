## Development

- **Dev server**: http://localhost:4322
- **Framework**: Astro 5.16
- **Node adapter**: Standalone mode for SSR

## Content Structure

- `/src/content/species/` - 84 JSON files with species data
- `/src/pages/species/` - Species listing and detail pages
- `/public/assets/img/parrots/` - Species images (WebP format)

## Pages

- `/` - Homepage with category overview
- `/species` - Complete species listing with search/filter
- `/species/[slug]` - Individual species detail pages
- `/es/` - Spanish language versions

## Site Information

- **Domain**: parrotspeciesguide.com
- **Content**: Extracted from Find Your Parrot marketplace
- **Purpose**: Dedicated species information resource
